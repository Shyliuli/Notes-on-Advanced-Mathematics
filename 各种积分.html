Part 1: 二重积分与三重积分 (基础)

核心思想

  • 二重积分 \(\iint_D f(x,y) \,d\sigma\): 求曲顶柱体的体积。
  • 三重积分 \(\iiint_\Omega f(x,y,z) \,dV\): 求空间物体的质量 (当 \(f\) 为密度时)。若 \(f=1\)，则为求体积。

计算方法：化为定积分

二重积分 \(\iint_D f(x,y) \,dxdy\)

  • 直角坐标: "穿线定限"。先y后x: \(\int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y) \,dy\)。
  • 极坐标: 适用于积分域为圆、扇、环或被积函数含 \(x^2+y^2\)。
    \[ \iint_D f(x,y) \,dxdy = \iint_{D^*} f(r\cos\theta, r\sin\theta) \, \color{red}{r} \,drd\theta \] 关键易错点：千万别忘了乘雅可比行列式 \(r\)！

三重积分 \(\iiint_\Omega f(x,y,z) \,dV\)

  • 投影法 (直角坐标): 先对z积分，再对xy平面投影。 \[ \iiint_\Omega f \,dV = \iint_{D_{xy}} \left[ \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z) \,dz \right] \,dxdy \]
  • 柱面坐标: \(x=r\cos\theta, y=r\sin\theta, z=z\)。适用于圆柱、圆锥体。
    \[ dV = \color{red}{r} \,drd\theta dz \]
  • 球面坐标: \(x=\rho\sin\varphi\cos\theta, y=\rho\sin\varphi\sin\theta, z=\rho\cos\varphi\)。适用于球体。
    \[ dV = \color{red}{\rho^2\sin\varphi} \,d\rho d\varphi d\theta \]

Part 2: 曲线积分

第一类曲线积分 (对弧长)

  • 形式与意义: \(\int_L f(x,y) \,ds\)，求曲线段的质量。与路径方向无关。
  • 计算方法 (参数化): 设 \(L\) 的参数方程为 \(x=x(t), y=y(t) \quad (\alpha \le t \le \beta)\)。
    \[ \int_L f(x,y) \,ds = \int_\alpha^\beta f(x(t), y(t)) \sqrt{x'(t)^2 + y'(t)^2} \,dt \]

第二类曲线积分 (对坐标)

  • 形式与意义: \(\int_L P(x,y)dx + Q(x,y)dy\)，求变力沿路径做的功。与路径方向有关。
  • 计算方法 (参数化):
    \[ \int_L Pdx + Qdy = \int_\alpha^\beta \left[ P(x(t),y(t))x'(t) + Q(x(t),y(t))y'(t) \right] \,dt \]

Part 3: 曲面积分

第一类曲面积分 (对面积)

  • 形式与意义: \(\iint_\Sigma f(x,y,z) \,dS\)，求曲面的质量。与曲面哪一侧无关。
  • 计算方法 (投影法): 设曲面 \(\Sigma: z=z(x,y)\)，投影到 \(D_{xy}\)。
    \[ \iint_\Sigma f \,dS = \iint_{D_{xy}} f(x,y,z(x,y)) \sqrt{1 + (\frac{\partial z}{\partial x})^2 + (\frac{\partial z}{\partial y})^2} \,dxdy \]

第二类曲面积分 (对坐标)

  • 形式与意义: \(\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy\)，求流体穿过曲面的流量。与曲面法向量方向 (哪一侧) 有关。
  • 计算方法 (投影法): "投一算一，符号看方向"。以 R 项为例：
    \[ \iint_\Sigma R(x,y,z) \,dxdy = \pm \iint_{D_{xy}} R(x,y,z(x,y)) \,dxdy \] 符号判断：法向量与z轴正向夹角为锐角(\(cos\gamma > 0\)，如上侧/外侧)取“+”，钝角(\(cos\gamma < 0\)，如下侧/内侧)取“-”。

Part 4: 格林公式 & 高斯公式 (降维打击)

格林公式 (Green's Theorem)

联系 平面闭合曲线积分 与 二重积分。

  • 条件: L为分段光滑的闭合曲线，方向为正向 (逆时针)。D为L围成的区域。
  • 公式:
    \[ \oint_L Pdx + Qdy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) \,dxdy \]
  • 技巧: 若曲线不闭合，可补线再减去。若为顺时针，结果加负号。

高斯公式 (Gauss's Divergence Theorem)

联系 空间封闭曲面积分 与 三重积分。

  • 条件: \(\Sigma\)为分段光滑的封闭曲面，方向取外侧。 \(\Omega\) 为\(\Sigma\)围成的空间区域。
  • 公式:
    \[ \oiint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iiint_\Omega \left( \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} \right) \,dV \]
  • 技巧: 若曲面不封闭，可补面再减去。若为内侧，结果加负号。
